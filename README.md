# eBookRelease
a tool for managing e-books written by C# 2.0

随着用户电子书的数量越来越多，采用传统的文件目录是管理已经难以满足人们的需要，查找不方便。所以本软件专门针对电子书管理开发，可以方便的管理，查找，统计，输出电子书列表等功能。简单实用，附有使用的详细说明文档。在.NET2.0 下开发，需要.NET Framework2.0支持
